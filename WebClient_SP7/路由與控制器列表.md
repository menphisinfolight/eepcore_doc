# EEP Core WebClient 路由與控制器（SP7）

> 原始碼位置：`EEPWebClient.Core/`
> 路由設定：`Startup.cs` line 147–161
> 路由轉換器：`Utility/RouteTransformer.cs`

## 路由總覽

EEP Core 使用 ASP.NET Core 的 Dynamic Route 機制，由 `RouteTransformer` 將 URL 路徑轉換到對應的 Controller/Action。

### 動態路由對照表

| URL 模式 | RouteTransformer | Controller | Action | 說明 |
|----------|-----------------|------------|--------|------|
| `bootstrap/{page}` | RWDRouteTransformer | **Main** | RenderPage | RWD 頁面渲染（前台） |
| `design/{page}` | DesignRouteTransformer | **Design** | RenderPage | 設計介面頁面 |
| `userdesign/{fileType}/{fileName}` | DownloadFileRouteTransformer | **UserDesign** | DownloadFile | 檔案下載 |
| `userdesign/{fileType}/backup/{fileName}` | DownloadFileRouteTransformer | **UserDesign** | DownloadFile | 備份檔案下載 |
| `flow/{page}` | FlowRouteTransformer | **Flow** | RenderPage | 流程頁面 |
| `method/{name}` | MethodRouteTransformer | **Main** | CallMethod | 呼叫自訂方法 |
| `proc/{name}` | ProcRouteTransformer | **Main** | CallProc | 呼叫 Processor |
| `captcha.png` | CaptchaTransformer | **Logon** | RenderCaptcha | 驗證碼圖片 |
| `{route}` | UserRouteTransformer | **Main** | RoutePage | 通用路由（fallback） |
| `{controller}/{action}/{id?}` | — | 依 URL | 依 URL | 預設 MVC 路由 |

### 路由匹配順序

```
1. bootstrap/{page}     → RWD 頁面
2. design/{page}        → 設計介面
3. userdesign/...       → 檔案下載
4. flow/{page}          → 流程頁面
5. method/{name}        → 自訂方法
6. proc/{name}          → Processor
7. captcha.png          → 驗證碼
8. {controller}/{action} → 標準 MVC（Home/Account/File 等）
9. {route}              → 通用 fallback
```

## 請求處理流程

```
HTTP Request
  → Middleware（Session、Auth、CORS）
  → Dynamic Route Matching
    → RouteTransformer 決定 Controller/Action
  → Controller Action 執行
    → 頁面渲染（RenderPage）
    → 或資料操作（GetDataset/UpdateDataset/CallMethod）
  → HTTP Response
```

### RWD 頁面渲染流程（最常見）

```
GET /bootstrap/OrderManage
  → RWDRouteTransformer → Main.RenderPage
  → 讀取 design/client/{Solution}/OrderManage.json
  → RWDPage.LoadControls() 載入元件
  → RWDPage.Render() 產生 HTML
  → 回傳 HTML 頁面
```

### 資料操作流程

```
POST /main（AJAX）
  → MainController.Index()
  → 根據 param["type"] 分派：
    → "data" → DataProvider（查詢/更新/匯出）
    → "security" → SecurityProvider（權限管理）
    → "design" → DesignProvider 系列
    → "flow" → FlowProvider（流程操作）
    → "log" → LogProvider（日誌）
    → ...
```
